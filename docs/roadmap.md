# Roadmap

## Milestones

1. **Editor scaffold + upload**
   - Build foundational project model and editor shell.
   - Ship asset upload flow with format validation and project attachment.

2. **Template engine + ranking/scoreboard packs**
   - Implement template rendering/application engine.
   - Release initial template packs focused on ranking graphics and scoreboards.

3. **Export pipeline**
   - Build rendering/export service path with status tracking.
   - Deliver downloadable outputs and robust failure reporting/retry hooks.

4. **Collaboration/versioning and advanced automation**
   - Add project history/version controls and collaborative workflows.
   - Introduce advanced automation for repetitive graphics production tasks.
