# Product Spec

## Core User Persona
**Short-form creators** who produce fast-turnaround content for social platforms and need polished on-screen graphics (titles, scoreboards, stat callouts, lower thirds) without complex motion-design tooling.

## Primary Jobs-to-be-Done
1. **Get from raw media to branded graphics quickly** so creators can publish while trends are still relevant.
2. **Apply consistent visual identity** across videos without rebuilding layouts each time.
3. **Edit text and key visual properties fast** during iteration (name changes, scores, colors, timing).
4. **Export reliably in platform-ready formats** with minimal troubleshooting.

## MVP Feature List
- **Upload**
  - Upload media assets (video/image/logo) and basic project files.
  - Validate supported formats and provide clear error messaging.
- **Templates**
  - Start from reusable graphic templates for common creator use cases.
  - Include starter sets for titles, lower thirds, and simple scoreboards.
- **Edit**
  - In-browser editor for text, colors, positioning, timing, and basic layer visibility.
  - Real-time preview with autosave.
- **Export**
  - One-click export to common creator-friendly output settings.
  - Download exported assets with visible progress and completion state.

## Non-Goals for MVP
- Full professional motion-graphics parity with advanced timeline/keyframe systems.
- Deep multi-user real-time collaboration.
- Marketplace/ecosystem for third-party template sales.
- AI-assisted generation workflows beyond basic template usage.
- Enterprise governance features (SSO, audit tooling, granular permissions).

## Success Metrics
- **Time-to-first-export**: Median time from project creation to first successful export.
- **Export success rate**: Percentage of export attempts that complete successfully without user intervention.
